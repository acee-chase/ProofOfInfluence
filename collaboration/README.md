# AI 协作开发系统

> ChatGPT + Cursor + Replit + 人工的4方协作模式

**适用项目**: 任何使用 React/TypeScript 的 Web 项目  
**核心理念**: AI辅助开发 + 人工决策 + 自动化部署

## 📊 4方角色分工

| 工具 | 角色 | 时间占比 | 核心职责 |
|------|------|---------|---------|
| **ChatGPT** | 战略规划师 | 15% | 需求分析、架构设计、问题诊断 |
| **Cursor** | 编码执行官 | 70% | 代码实现、调试重构、本地测试 |
| **Replit** | 部署运维官 | 5% | 自动部署、环境管理、日志监控 |
| **人工** | 决策指挥官 | 10% | 质量把关、流程控制、效果评估 |

---

## 🔄 基本工作流

```
开始工作
  ↓
查看 DAILY_CHECKLIST.md
  ↓
ChatGPT 规划 (5-10分钟)
  ↓
Cursor 开发 (30-60分钟)
  ↓
本地测试 (10-15分钟)
  ↓
Git 提交 → GitHub
  ↓
Replit 自动部署 (2分钟)
  ↓
生产验证
```
